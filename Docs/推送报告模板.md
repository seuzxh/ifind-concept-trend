# 推送报告模板

企业微信群机器人 Webhook 推送格式定义。

## 接口说明

- **请求方式**：`POST {webhook_url}`
- **Content-Type**：`application/json`
- **消息类型**：`msgtype=markdown`
- **字节限制**：Markdown 内容不超过 4096 字节

## 请求体格式

```json
{
    "msgtype": "markdown",
    "markdown": {
        "content": "<markdown_content>"
    }
}
```

## 报告模板

```
### 概念板块强势扫描 {date}

> 扫描时间：{scan_time} | 观察股池：{pool_count}只 | 强势个股：{strong_count}只

**Top 10 强势个股（按板块归类）**

▎{板块名} — 板块得分 {board_score}
  {全局排名}. **{股票名}({code})** 得分{score} | 涨{change}% | 实{body}% | {amount}
  {全局排名}. {股票名}({code}) 得分{score} | 涨{change}% | 实{body}% | {amount}
  ...

▎{板块名} — 板块得分 {board_score}
  ...

**Top 5 强势板块**

▎{rank}. {板块名} | 得分 {board_score} | 强势 {strong}/{total}
  1. **{股票名}({code})** 得分{score} | 涨{change}% | 实{body}% | {amount}
  2. {股票名}({code}) 得分{score} | 涨{change}% | 实{body}% | {amount}
  ...
  5. ...

▎{rank}. {板块名} | 得分 {board_score} | 强势 {strong}/{total}
  1. ...
  5. ...
```

### 两段式结构说明

| 区域 | 内容 | 作用 |
|------|------|------|
| **第一段：Top 10 个股归类** | 全局得分 Top 10 个股，按归属板块分组 | 快速定位最强个股 |
| **第二段：Top 5 板块详情** | 每个板块展示其 Top 5 成分股 | 了解板块内部结构 |

### 归类与归属规则

**第一段（Top 10 个股归类）**：
1. 个股按 `score` 降序取 Top 10
2. 每只个股归入其所属概念板块中 **board_score 最高** 的板块（避免重复）
3. 板块按 `board_score` 降序排列，板块内个股按 `score` 降序
4. 全局编号 1~10 保持连续

**第二段（Top 5 板块详情）**：
1. 板块按 `board_score` 降序取 Top 5
2. 每个板块展示其全部归属个股的 Top 5（按 `score` 降序）
3. 个股归属逻辑同上：取所属概念板块中 board_score 最高的板块
4. 每个板块内编号 1~5 独立编号

### 格式规则

| 规则 | 说明 |
|------|------|
| 强势个股加粗 | `is_strong=1` 的个股名称用 `**加粗**` 显示 |
| 涨幅符号 | 正数前加 `+`，如 `+9.39%` |
| 成交额单位 | 超亿显示 `X.XX亿`，否则 `XXXX万` |
| 得分保留两位 | 如 `86.07` |
| 板块名截断 | 最长 12 字符，超出截断加 `...` |
| 板块分隔符 | `▎` 符号标记板块分组头 |

## 模板变量映射

### 概览区域

| 模板变量 | 数据来源 | 格式化规则 |
|---------|---------|-----------|
| `{date}` | trade_date | YYYY-MM-DD |
| `{scan_time}` | 当前时间 | HH:MM |
| `{pool_count}` | len(stock_results) | 整数 |
| `{strong_count}` | sum(is_strong) | 整数 |

### 第一段：Top 10 个股归类

| 模板变量 | 数据来源 | 格式化规则 |
|---------|---------|-----------|
| `{板块名}` | 个股归属板块的 concept_name | 最长 12 字符 |
| `{board_score}` | 归属板块的 board_score | 保留 2 位 |
| `{全局排名}` | 个股在 Top 10 中的排名 | 1~10 |
| `{股票名}` | stock_results[n].stock_name | 原始 |
| `{code}` | stock_results[n].stock_code | 去掉后缀(.SZ/.SH) |
| `{score}` | stock_results[n].score | 保留 2 位 |
| `{change}` | stock_results[n].change_ratio | 正数加 +，保留 2 位 |
| `{body}` | stock_results[n].body_change_ratio | 正数加 +，保留 2 位 |
| `{amount}` | stock_results[n].total_amount | 亿/万 格式化 |

### 第二段：Top 5 板块详情

| 模板变量 | 数据来源 | 格式化规则 |
|---------|---------|-----------|
| `{rank}` | 板块排名 | 1~5 |
| `{板块名}` | board_results[n].concept_name | 最长 12 字符 |
| `{board_score}` | board_results[n].board_score | 保留 2 位 |
| `{strong}/{total}` | board_results[n] | N/M |
| 成分股行 | 同上"成分股行" | 同上 |

### 成交额格式化规则

```
if amount >= 1亿:    "X.XX亿"    (如 1.23亿)
elif amount >= 1万:  "XXXX万"    (如 8500万)
else:                "X元"       (如 9800元)
```

### 个股板块归属算法

```python
# 1. 建立 board_score 排名字典
board_rank = {
    board["concept_name"]: board["board_score"]
    for board in board_results
}

# 2. 每只个股归入其所属板块中得分最高的
for stock in stock_results:
    concepts = stock["concept_names"].split(",")
    best_concept = max(
        concepts,
        key=lambda c: board_rank.get(c, 0),
    )
    stock["best_concept"] = best_concept

# 3. 第一段：取 score Top 10 个股，按归属板块分组
top_10 = sorted(stock_results, key=lambda s: s["score"],
                reverse=True)[:10]

# 4. 第二段：每个板块取其归属个股的 Top 5
for board in top_5_boards:
    board_stocks = [
        s for s in stock_results
        if s["best_concept"] == board["concept_name"]
    ]
    board_stocks.sort(key=lambda s: s["score"],
                      reverse=True)
    board["top_stocks"] = board_stocks[:5]
```
