# 涉及 ifind 接口整理

## 一、鉴权token获取和校验
1. 根据 `refresh_token` 获取 `access_token`
2. 完成鉴权校验
3. 能够查询所剩额度
4. 每天任务开始前，检查额度是否足够，若不足则需要重新获取token

概念人气明细选择标准
选择t-1日 自选热度 Top20、自选热度变化率 Top20 概念板块

概念人气股挑选：
1. 选择t-1日 概念板块内 自选热度 Top20、自选热度变化率 Top20 涨幅 股票
2. 剔除ST股

**概念板块和概念股需要去重作为T日的监控标的**



## 二、获取概念人气明细和成分股信息接口

### 1. 获取某一日期概念人气明细接口
以20260602为例

```
#概念人气明细,输入参数:交易日期(date)、统计周期(tjzq)-iFinD数据接口
requestMethod:POST
requestURL:https://quantapi.51ifind.com/api/v1/data_pool
requestHeaders:{"Content-Type":"application/json","access_token":"a9211df9b915af20a893560e042a17e71f3563d2.signs_NjM0ODM2NDI5"}
formData:{"reportname":"p03797","functionpara":{"date":"20260602","tjzq":"近一周"},"outputpara":"p03797_f001,p03797_f002,p03797_f009,p03797_f010"}
```
- `p03797_f009` : 自选热度
- `p03797_f010` : 自选热度变化率（%）

以“芯片概念“为例

```

requestMethod:POST
requestURL:https://quantapi.51ifind.com/api/v1/data_pool
requestHeaders:{"Content-Type":"application/json","access_token":"a9211df9b915af20a893560e042a17e71f3563d2.signs_NjM0ODM2NDI5"}
formData:{"reportname":"p03798","functionpara":{"date":"20260602","tjzq":"近一周","hy":"芯片概念"},"outputpara":"jydm,jydm_mc,p03798_f001,p03798_f012,p03798_f016"}
```

- `p03798_f012` : 涨跌幅（%）
